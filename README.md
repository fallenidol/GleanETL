GleanETL
========
[![Build status](https://ci.appveyor.com/api/projects/status/wcgnmg6ob67t69aw/branch/master?svg=true)](https://ci.appveyor.com/project/fallenidol/gleanetl/branch/master)

Makes extracting data from text files that are in difficult formats a snap. Aims to provide a flexible [ETL](https://en.wikipedia.org/wiki/Extract,_transform,_load) framework with a focus on performance.
